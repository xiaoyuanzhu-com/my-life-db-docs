The full REST API surface is documented as an [OpenAPI 3.1 spec](/docs/openapi.yaml) and rendered with [Scalar](https://scalar.com/).

[Open the interactive reference →](/docs/api/reference/)

The reference covers the **current baseline** — every endpoint as it exists today, before the namespace refactor planned in the API ADRs. It includes:

- All authentication flows (session cookie, OAuth, Connect bearer)
- File library operations (read/write/move/pin)
- Full-text search
- Notifications (SSE)
- Settings and stats
- Collectors and apps
- Connect (OAuth 2.1 PKCE) for third-party apps
- Agent sessions (REST + WebSocket)
- Explore feed
- Sharing
- MCP transport
- Raw and SQLAR file serving
