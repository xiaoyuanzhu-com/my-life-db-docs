The Inbox is your primary capture space. Drop in text, files, photos, links, voice memos — anything — without deciding where it belongs. Capture now, organize later.

## How It Works

The Inbox is a chat-style feed. Newest items appear at the bottom, older items are above. Scroll up to load more history, or just let new items flow in at the bottom.

Everything you add is a real file on disk. There's no proprietary format — your data stays yours.

## Adding Content

Type text into the input bar at the bottom and press Enter. You can also attach files:

- **Drag and drop** files from your desktop
- **Click the attachment button** to pick files
- **Paste images** directly from your clipboard

Multiple files can be sent together with optional text. Large uploads are resumable — if your connection drops, the upload picks up where it left off.

## What Happens After Upload

Files are automatically enriched by AI processing in the background:

- **Images** get captions, OCR text extraction, and object detection
- **Documents** (PDF, Office) get converted to markdown and a visual screenshot
- **Audio** gets transcribed with speaker identification
- **Links** get crawled and summarized
- **Everything** gets tagged and indexed for search

You can see processing status on each item and re-trigger enrichment if needed.

## File Cards

Each item renders as a type-specific card:

| Type | What You See |
|------|-------------|
| **Text / Markdown** | Prose preview (up to 20 lines) |
| **Image** | Inline thumbnail |
| **Video** | Thumbnail with play button |
| **Audio** | Waveform with play icon |
| **PDF / Office docs** | Document icon with filename and size |

Tap any card to open a full-screen preview.

## Full-Screen Preview

The preview viewer adapts to the file type — images get pinch-to-zoom and swipe navigation, videos get playback controls, PDFs render with continuous scroll, and text files show in a readable monospace font.

For images and videos, swipe left/right to page through your media. The media pager follows chronological order — swipe left for newer, right for older.

## Pinning

Pin important items to keep them accessible. Pinned items appear in a horizontal bar above the input. Tap a pinned item to jump directly to it in the feed.

Pin or unpin from the context menu (right-click or long-press on any card).

## Search

Type a search query to find items by content. MyLifeDB uses both keyword and semantic search — so searching for "vacation photos" can find images even if they aren't literally labeled that way.

Search results replace the feed. Use "Locate in Feed" on any result to jump to its position in the timeline.

## Multi-Select

Right-click or long-press and choose "Select" to enter multi-select mode. Checkboxes appear on every card. Select items, then use the action bar to share or delete in bulk.

## Real-Time Updates

The Inbox updates live. If you add a file from another device or through the filesystem directly, it appears in the feed within about a second — no manual refresh needed.
